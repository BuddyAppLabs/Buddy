# Project Rules

## Communication & Comments
- Use Chinese for comments and docs

## Package Manager
- `pnpm`

## Import Rules
- Use `@` alias for imports
- Example: `import SearchBar from '@/ui/layout/SearchBar.vue'`

## Frontend
- Vue3, `<script>` above `<template>`
- DaisyUI + TailwindCSS, no hardcoded colors
- Icons under `icons`

## Electron
- See `packages/buddy/package.json` and `electron.vite.config.mts`
