# 开发指南

本节包含 Buddy 应用的开发相关文档，帮助开发者理解和扩展应用功能。

## 窗口系统

### [双窗口模式](/manuals/zh-cn/dev/window-modes)
了解如何使用和配置精简模式（快捷键悬浮窗）和完整模式（传统窗口）。

**主要内容：**
- 两种窗口模式的特点和使用场景
- 配置方法和 API 文档
- 前端适配指南
- 技术实现细节

## 快速开始

如果你是第一次开发 Buddy，建议按以下顺序阅读：

1. [架构概览](/manuals/zh-cn/architecture) - 了解整体架构
2. [插件开发](/manuals/zh-cn/plugins) - 学习如何开发插件
3. [双窗口模式](/manuals/zh-cn/dev/window-modes) - 理解窗口系统

## 开发环境

确保你已经安装了以下工具：

- Node.js 18+
- pnpm
- Electron
