## 核心概念

### 主进程 (Main Process)
- **职责**：应用核心逻辑、系统服务、插件管理
- **入口**：`packages/buddy/src/main/bootstrap/app.ts`
- **技术栈**：Node.js + Electron + TypeScript

### 渲染进程 (Renderer Process)
- **职责**：用户界面、交互逻辑
- **入口**：`packages/buddy/src/ui/main.ts`
- **技术栈**：Vue 3 + Tailwind CSS + DaisyUI

### 插件系统 (Plugin System)
- **职责**：动态扩展应用功能
- **加载方式**：进程内直接调用（通过 `import()` 动态加载）
- **通信方式**：通过 `SuperContext` 注入内核能力
